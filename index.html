Energy-Efficient Smart Home Automation Using ESP32, MQTT and AI-Based Voice Command Processing

Low-cost intelligent smart home automation system integrating MQTT communication, AI-assisted voice control, real-time dashboard synchronization and embedded control engineering.

View Repository

System Overview

This project presents a scalable and intelligent smart home automation platform designed using ESP32 microcontrollers, MQTT communication architecture and AI-assisted voice interaction technologies.


The system supports:


  • Real-time MQTT device synchronization
  • AI-based voice-assisted control
  • Manual local override operation
  • Web dashboard monitoring and control
  • Environmental telemetry monitoring
